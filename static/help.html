Comment mettre une émission en ligne ?

Loading the player...
  1. Télécharger FileZilla
  2. Installez et lancez FileZilla (sans blague... je croyais que c'était pour décorer mon bureau)
  3. Vous devez arriver sur la page de démarrage du logiciel. Dans la fenêtre de gauche, vous voyez les fichiers présents sur votre ordinateur, et à droite, les fichiers présents sur le serveur (quand vous serez connecté).
  4. Cliquez sur le "Gestionnaire de Sites". Cliquez sur "Nouveau Site", appelez le par exemple "onzerocks" et remplissez les champs avec les informations transmises par mail.Voir la page.
  5. Cliquez sur "Connexion". Dans la colonne de droite devrait apparaître l'arborescence du serveur.
  6. Dans la colonne de droite (côté serveur), allez dans "www -> onzerocks -> store -> audio".
  7. Trouvez le fichier à transférer sur votre ordinateur. Vous pouvez le trouver :
    • Soit avec FileZilla (dans la colonne de gauche). Dans ce cas, il faudra double cliquer dessus pour le transférer, ou bien le faire glisser de la colonne de gauche vers la colonne de droite.
    • Soit avec votre explorateur de fichier habituel. Dans ce cas, il faut le faire glisser dans la colonne de droite de FileZilla.
  8. Une fois le transfert entamé, on peut surveiller sa progression dans la zone de transfert en bas.
  9. Quand le transfert est terminé, allez sur le site, connectez vous, allez dans l'onglet "Gérer les émissions", cliquez sur "Ajouter".
  10. Le fichier que vous venez de tranférer devrait apparaître. Cliquez dessus puis remplissez le formulaire.
  11. Et voilà, l'émission est en ligne !
